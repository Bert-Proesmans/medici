Build a safe wrapper from a UnsafeTrigger object.

Safety

The exact StateContainer (state machine) is removed from the UnsafeTrigger. A safe wrapper can thus be generated for ANY state machine which TriggerState-associated types Timing and Trigger match on the requested machine.

See TriggerService for more information!
