Crate game_system[−][src]

Example crate

Implements the state machine for a board game. Any type which interacts with the built state machine MUST be exported from this crate. Any type that's directly used from medici_core MUST transitively be re-exported here. Re-exporting allows the downstream crates to have a single dependancy, this crate.

See also

[game_rules] for an example how to implement game rules which use the state machine defined within this crate.
